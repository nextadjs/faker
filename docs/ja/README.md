# @nextad/faker

OpenRTB/Creative/AdCOMのモックデータを生成するTypeScriptフェイカーライブラリ。

## 特徴

- 🎯 TypeScript型定義完全サポート
- 🔄 直感的なメソッドチェーンによるオブジェクトの構築
- 📝 OpenRTB/AdCOM/Native-Ad/VAST 規格準拠
- ⚡ ESMとCommonJSサポート

## インストール

npm:
```bash
npm install iab-openrtb iab-openrtb iab-native
npm install -D @nextad/faker
```

pnpm:
```bash
pnpm add iab-openrtb iab-openrtb iab-native
pnpm add -D @nextad/faker
```
